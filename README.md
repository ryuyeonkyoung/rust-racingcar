# rust-racingcar-precourse

### 입력
- [x] 경주할 자동차 이름(이름은 쉼표(,) 기준으로 구분)를 입력받는다.
    - 예외:
        - [] 자동차 이름이 이름은 5자 이하가 아닌 경우
        - [] 한 대의 자동차만 입력된 경우
        - [] 같은 자동차 이름이 있을 경우
        - [] 구분자 뒤에 값이 없을 경우
- [x] 시도할 횟수를 입력받는다.
    - 예외:
        - [] 시도할 횟수에 정수가 아닌 수가 들어간 경우

### 자동차 경주
- [x] 주어진 횟수 동안 n대의 자동차의 전진 또는 멈춤 여부를 결정한다.
    - [x] 전진하는 조건은 0에서 9 사이에서 무작위 값을 구한 후 무작위 값이 4 이상일 경우이다.
- [x] 전진 또는 멈춤 여부에 따라 차를 움직인다.

### 출력
- [x] 차수별 실행 결과를 출력한다.
- [x] (단독 또는 공동) 우승자 안내 문구를 출력한다.
    - [x] 우승자가 여러 명일 경우 쉼표(,)를 이용하여 구분한다.

### 구조체
- `Car`
- `Cars`
